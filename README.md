# Samnetic 品牌介紹網站

這是一個為 **Samnetic** 品牌打造的單頁式介紹網站，展現我們全方位科技形象、創新精神與品牌定位。

## 🔧 功能特色

- 🌟 動態粒子背景動畫（HTML5 Canvas）
- 🎯 吸引人的 CTA 按鈕滑動效果
- 📄 品牌優勢說明區塊
- 📨 聯絡我們表單（Formspree 支援）
- 🌈 溫暖淡紫配色，專業親切兼具
- 📱 響應式設計，支援桌機與手機
